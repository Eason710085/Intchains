环境配置：
1. 安装nodejs， 本项目使用的npm版本为6.14.13，node版本为14.17.3
2. 安装飞冰框架：https://alibaba.github.io/ice/ 或在Vscode中安装AppWorks组件
3. 使用Fushion框架
4. 安装ProComponents 组件 npm i @ant-design/pro-table --save
5. mkdir 新文件夹
6. cd 新文件夹
7. 使用本项目代码文件替换掉初始化生成的文件
8. npm install
9. npm run start
